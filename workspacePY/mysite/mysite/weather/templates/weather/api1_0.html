http://openweatherdata.ns0.it:8000/weather/api1_0/history/

http://openweatherdata.ns0.it:8000/weather/api1_0/city_list/

To get single data by datetime
http://openweatherdata.ns0.it:8000/weather/api1_0/getSingleData/
Params: name,dt (example dt='2017-04-12')
To get last temperature
http://openweatherdata.ns0.it:8000/weather/api1_0/getLastTempData/
Params: name (example name=Palermo)
To get last humidity
http://openweatherdata.ns0.it:8000/weather/api1_0/getLastHumyData/
Params: name (example name=Palermo)
To get last pressure
http://openweatherdata.ns0.it:8000/weather/api1_0/getLastPressure/
Params: name (example name=Palermo)
To get single data by datetime RDF ontology
http://openweatherdata.ns0.it:8000/weather/api1_0/getSingleDataRDF/
Params: name,dt (example dt='2017-04-12')
To get history by city RDF ontology
http://openweatherdata.ns0.it:8000/weather/api1_0/historyRDF/
Params: name (example name=Palermo)
