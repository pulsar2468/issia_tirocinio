What is it??On the right, we have the heart of the anemometer (tcrt5000). It's an reflective sensor which include an infrared emitter and phototransistor in a leaded package which block visible light. The pin D1 listens the current flow in the phototransistor and If the infrared wave is reflected a change in the current flow occurs. The resistors are recommended by the manufacturer for optimal operation.
Tangential Speed $$v=\frac{2\pi r}{T} \implies v=\frac{d\pi}{T} \\ \hspace6ex v=\frac {m}{s} \hspace2ex T=Period \\r=radius \\2r = diameter $$ Since the formula expresses the speed in meters per second, you need multiplication the result with 3.6. In this way, we obtain the result converts into km/h For example: $$ \frac {25m}{s} = \frac {25\frac {1km}{1000m}}{\frac {1h}{3600s}}=90km/h $$
Explanation Since the infrared wavelength is between 700 nm and 1 mm (is longer than visible light, 400 nm-700nm) The wave of the emitter infrared will be reflected only by object with same wavelength, in this case white object. We are interest to have two states in way to obtain a dualistic concept. By this paragon, we exploit the boolean logic 1 o 0 depending if the wave has been reflected or not. A infrared wave can be reflected only by white object or similar (Silver object) instead, it's won't be reflect by black material
